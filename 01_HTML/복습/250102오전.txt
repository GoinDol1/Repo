- 테이블을 표현하는 태그 : table
- 제목 셀을 표현하는 태그 : th
- 내용 셀을 표현하는 태그 : td

- 테이블에서 제목 부분을 구분하는 태그 : thead
- 테이블에서 내용 부분을 구분하는 태그 : tbody
- 테이블에서 요약/정리 부분을 구분하는 태그 : tfoot

- 다수의 열을 병합하고자 할 경우 사용되는 속성 : colspan
- 다수의 행을 병합하고자 할 경우 사용되는 속성 : rowspan
